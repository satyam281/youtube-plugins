## HOW TO LOAD INTO BROWSER

### 1. Download zip 
### 2. Unzip as a folder 
### 3. Go to chrome://extensions on your chromium based browser
### 4. Toggle developer mode on from top right 
### 5. Select load unpacked -> Select the unzipped folder
